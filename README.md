# StudySphere

## Overview
StudySphere is an AI-powered learning platform designed to help students organize study materials, generate quizzes, manage notes, and improve learning efficiency.

## Features
- User Authentication
- Google Login
- AI Quiz Generation
- Notes Management
- Subject-wise Content Organization
- Dashboard Analytics
- Online Study Resources

## Tech Stack
Frontend: HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: Supabase
Authentication: Supabase Auth

## Installation
1. Clone the repository
2. Install dependencies
3. Configure environment variables
4. Run the server
